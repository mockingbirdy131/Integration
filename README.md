# Integration

Данный проект реализует численное интегрирование функции на отрезке с использованием составных квадратурных формул различного порядка. Программа позволяет сравнить эффективность различных методов численного интегрирования и оценить их погрешности.

В программе реализуются следующие шаги:

1. Задание параметров задачи, а также функции и ее первообразной:
   - Концы отрезка [a, b]
   - Количество отрезков K, на которые разбивается отрезок [a, b]
   - Исходная функция (func)
   - Аналитически вычисленная первообразная (abs_int)

2. Построение сетки:
   - Равномерное разбиение отрезка [a, b] на K и 2K отрезков
   - Шаг интегрирования h = (b − a) / K и h = (b − a) / (2K)

3. Численное интегрирование (5 методов):
   - Составная формула прямоугольников
   - Составная формула трапеций
   - Составная формула Симпсона
   - Составная формула Ньютона-Котеса (5 узлов)
   - Составная формула Гаусса (3 узла)

4. Анализ погрешности:
   - Вычисление относительной погрешности для каждого метода
   - Сравнение результатов при разном количестве отрезков (K и 2K)
